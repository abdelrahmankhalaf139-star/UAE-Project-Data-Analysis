# UAE Infrastructure Project Analysis

## Overview

This project presents an exploratory data analysis (EDA) of infrastructure projects across the United Arab Emirates. Using Python and data visualization libraries, the analysis explores project distributions, industry trends, statistical summaries, and correlations between project value and completion year.

The project demonstrates a complete data analysis workflow, including data loading, preprocessing, descriptive statistics, and visualization.

---

## Features

- Data loading using Pandas
- Exploratory Data Analysis (EDA)
- Descriptive statistics
- Distribution analysis
- Correlation analysis
- Industry-specific filtering
- Data visualization with Matplotlib and Seaborn

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---



## Dataset

The dataset contains information about infrastructure projects across the UAE, including:

- Project ID
- Project Name
- City
- Industry
- Sector
- Project Value (Million USD)
- Project Status
- Completion Year
